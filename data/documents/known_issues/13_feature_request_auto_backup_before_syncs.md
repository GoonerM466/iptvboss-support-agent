# Feature Request: Auto-Backup Before Syncs

Request for automatic database backup before risky operations like source syncs.

### Current Limitation

Manual backup required before risky operations.

### Feature Request

Automatic backup before source syncs (with retention policy).

### Benefit

Protection from provider changes breaking layouts. Easy recovery from sync problems without remembering to backup manually.
