# Importing Channels into Layouts - Overview

Once you have sources configured, you need to import channels from those sources into your layouts. IPTVBoss provides two dialogs for this purpose.


**Visual References**:
- [image13](https://goonerm466.github.io/iptvboss-support-agent/images/image13.png) - Import Channels From Sources dialog showing source selection, categories list, and channels list with import options
- [image17](https://goonerm466.github.io/iptvboss-support-agent/images/image17.png) - Import Channels From Layouts dialog showing layout selection, groups list, and channels list with import options
**Access**: In Layout Editor, click the **Import Channels** button (often labeled "Add Channels")
