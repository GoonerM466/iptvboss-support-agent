# TinyURL Authentication Errors (Unauthenticated or 503)

TinyURL API key errors or service availability issues.

### Problem

You're receiving one of these errors when trying to use TinyURL:
- **'Unauthenticated' error**: Your TinyURL API key is either not entered or incorrect
- **'503 Service Unavailable' error**: TinyURL's service is temporarily down

### Solution

**For 'Unauthenticated' error:**
1. Go to **Settings â€™ TinyURL**
2. Verify your API key is entered correctly (no extra spaces)
3. If needed, generate a new API key from tinyurl.com and re-enter it

**For '503 Service Unavailable' error:**
1. Check if tinyurl.com loads in your browser
2. Try again in a few minutes
3. As a workaround, use direct Dropbox/Google Drive URLs
